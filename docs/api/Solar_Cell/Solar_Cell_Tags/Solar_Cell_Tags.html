relatedFetch (line 364)

Does a lookup on the related table returning specified columns.

This more-optimized search algorithm is derived from...

http://www.pui.ch/phred/archives/2005/04/tags-database-schemas.html http://www.petercooper.co.uk/archives/000648.html http://www.bigbold.com/snippets/posts/show/32

SELECT p.* FROM posts_tags pt, posts p, tags t WHERE pt.tag_id = t.id AND ( t.name = '" + tags.uniq.join ('\' OR t.name=\'') + "' ) AND p.id=pt.post_id GROUP BY p.id HAVING COUNT(p.id) = " + tags.uniq.length.to_s

(intersection/and...)

SELECT * FROM sc_tags WHERE rel = 'sc_bookmarks' AND tag IN ('this','that','other') -- count = 3 GROUP BY rel_id HAVING COUNT(rel_id) = 3 -- count = 3

(union/or: drop the HAVING clause)

  • return: An array of rows with the requested tags.
  • access: public
