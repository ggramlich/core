withTags (line 405)

Fetch all the bookmarks with a specific tagset.

Optionally, you can limit to a specific username as well.

Technically, the method searches the sc_bookmarks_tags table, which joins itself back to this table (sc_bookmarks) for the "real" info.

  • return: The list of bookmarks.
  • access: public
array withTags (string|array $tags, [string $user_id = null], [string $order = null], [int $page = null])
  • string|array $tags: The tags to look up; may be a space-separated string of tags. Will "AND" these for an intersection of tags.
  • string $user_id: The optional user_id to look up; if null, will look at all users.
  • string $order: A custom ORDER clause.
  • int $page: The page number to return.
