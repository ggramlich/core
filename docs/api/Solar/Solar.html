		autoload (line 276)
	
 
	
	

A "sham" method; __autoload() does not work with static calls.


If the class name exists as a key in $config['Solar']['registry'],  that array element value will be used as the file path.  If not,  the class name will be turned into a file path by converting  all instances of '_' in the class name to DIRECTORY_SEPARATOR  (i.e., '/' on Unix and '\' on Windows).
