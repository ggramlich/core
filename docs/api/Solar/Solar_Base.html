		locale (line 177)
	
 
	
	

Looks up locale strings based on a key.


Uses the locale strings in the directory noted by $conf['locale'];  if no such key exists, falls back to the strings for the parent  class, and finally falls back to the Solar/Locale strings.

	
    
				
  • return: The locale string, or the original $key if no  string found.
