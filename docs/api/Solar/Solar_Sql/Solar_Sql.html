Class for connecting to SQL databases and performing common operations.

Example usage:

  1.  $opts = array(
    2. 

  2.      'class' => 'Solar_Sql_Driver_Mysql',
    3. 

  3.      'host'   => 'localhost',
    4. 

  4.      'user'   => 'pmjones',
    5. 

  5.      'pass'   => '********',
    6. 

  6.      'name'   => 'test'
    7. 

  7.  );
    8. 

  8.  
    9. 

  9.  $sql = Solar::object('Solar_Sql', $opts);
    10. 

  10.  
    11. 

  11.  // a command with placeholders
    12. 

  12.  $stmt = "SELECT * FROM some_table WHERE date >= :first AND date <= :last";
    13. 

  13.  
    14. 

  14.  // data for the placeholders
    15. 

  15.  $data = array(
    16. 

  16.      'first' => '1970-01-01',
    17. 

  17.      'last'  => '1979-12-31'
    18. 

  18.  )
    19. 

  19.  
    20. 

  20.  // retrieve a result set from the bound command and data
    21. 

  21.  $result = $sql->exec($stmt, $data);
    22. 

  22.  
    23. 

  23.  // fetch the first row of the result set and free the result
    24. 

  24.  $row = $result->fetch();
    25. 

  25.  unset($result);
    26. 

  26.  
    27. 

  27.  // equivalent:
    28. 

  28.  $row = $sql->fetchRow($stmt, $data);

bind (line 230)

Binds placeholder text in an SQL statement with quoted values.

E.g., if $text = "SELECT * FROM table WHERE user = :user" and your $data = array('user' => 'bolivar') then the bound result would be "SELECT * FROM table WHERE user = 'bolivar'".
